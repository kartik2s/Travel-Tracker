# World Travel Tracker 🌍

A full-stack travel tracking web app where users can add and track visited countries on a world map.

## Features
- Track visited countries
- Multiple user/family support
- Dynamic world map coloring
- PostgreSQL database integration
- EJS templating

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- HTML/CSS

## Run Locally

```bash
npm install
node index.js
